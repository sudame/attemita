Attemita

Attemitaとは、「会って、見た」人をメモ・共有するサービスです。

特徴

  • 簡単なフォーム
    • 「##」を付けた行はトピックタイトルとして扱われ、それ以外が地の文として使われる
  • チーム共有機能
    • リアルタイムで情報共有できる
    • 共同同時編集機能を付けたかったが間に合わなかった

使ったもの

  • Vue.js：全体的に使用
  • Nuxt.js：全体的に使用
  • Vuex：登録者の管理などに利用。ページ遷移が超高速！
  • TypeScript：型は神
  • Firebase：ユーザ認証・認可とデータベースとして使用

チャレンジしたこと

  • 紙面のようなデザインを目指して、色を完全にモノクロにした
  • チーム共有があるのでセキュリティ面をちょっと気にした
  • ページ遷移が高速になるように気を付けた
